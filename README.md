# Decentralized Ethereum App

## Smart Contract with Web Development

Developed in the truffle framework using react for the front-end, this app deploys a smart contract onto localhost. It allows users to bet ether on a "randomly" generated number. Core of this project taken from https://medium.com/@merunasgrincalaitis/the-ultimate-end-to-end-tutorial-to-create-and-deploy-a-fully-descentralized-dapp-in-ethereum-18f0cf6d7e0e

## Parts

- webpack to bundle (and dev server)
- babel to compile
- react for frontend
- truffle as ethereum development framework
- ganache for local development
- web3 to connect ABI with front-end
 - abi acquired through solcjs
- solidity for contracts & tests
